GitHub repo for the manuscript:

# Machines Prefer Humans as Literary Authors: Evaluating Authorship Bias in Large Language Models

Content:

* ```code```: the code used to obtain the questionnaire answers from each LLM, to perform the statistical analysis, and to create the plots
* ```data```: all the filled questionnaires (```final_questionnaires.csv```), the prompt used, the questionnaire form, and the three AI generated stories
* ```results```: reports and plots with all the results obtained (including statistical tests)
